# Session Memory — Alexinomia Website Project

## What Was Built

An immersive, interactive, single-page informational website about **alexinomia** — the psychological phenomenon of anxiety-driven avoidance of using personal names in communication.

## Website Sections

1. **Hero** — Animated gradient title, floating particles, scroll hint
2. **Pull Quote** — Key quote from Ditye et al. (2023)
3. **What Is Alexinomia?** — Definition, etymology, info cards
4. **Stats** — Animated counters (257 sources, 190 participants, 3 publications)
5. **Symptoms** — 6 expandable/clickable cards (anticipatory anxiety, name avoidance, relationship strain, physical discomfort, shame, attachment insecurity)
6. **Research Timeline** — One card per peer-reviewed paper (2023, 2024, 2025)
7. **Impact** — Romantic relationships, friendships, workplace, mental health
8. **Self-Check Quiz** — 7 reflection questions, scored into 4 bands (not diagnostic)
9. **Get Help** — CBT, attachment-informed therapy, psychoeducation
10. **References** — All citations with live PubMed/DOI links

## Key Research Facts Captured

- Alexinomia = inability to use personal names in conversation due to anxiety (not memory failure)
- Coined by researchers at Sigmund Freud Private University Vienna
- Strongly linked to social anxiety disorder and insecure attachment styles
- More intense in closer relationships; affects all communication forms
- Cross-cultural — documented across languages and demographics
- Treatment: psychotherapy, particularly CBT for social anxiety
